Présentation

Derniere mise à jour : 18 juin 2026 à 11h30 min

Ce mini cours s'adresse aux étudiants débutants en informatique souhaitant maîtriser les bases fondamentales du développement web. Il propose une liste assez vaste des balises HTML, expliquées pas à pas avec des exemples concrets d'utilisation. L'objectif est de vous donner une autonomie totale pour structurer n'importe quelle page web de manière propre et professionnelle.

Maîtriser ces balises est une étape indispensable pour créer des sites rapides, accessibles à tous et parfaitement optimisés pour le référencement Google (SEO). Pour faciliter votre apprentissage, le cours est structuré par catégories : structure, texte, listes, tableaux, liens, multimédia, formulaires, sémantique et métadonnées. À la fin de ce programme, vous disposerez d'une boîte à outils complète pour donner vie à vos propres projets en ligne.

  • Chapitre 1 : Les Balises de Structure

    (header, nav, main, section, article, aside, footer, figure, figcaption)

    (meta, link, style, script, base)

CHAPITRE 1 : LES BALISES DE STRUCTURE

← Retour au Sommaire

Ces balises définissent la structure fondamentale de toute page HTML.

1.1. La balise <!DOCTYPE html>

Utilité : Déclare que le document utilise HTML5 (la version la plus récente).

Emplacement : Tout en haut du document, avant la balise <html>.

Exemple :

<!DOCTYPE html>
<html>
    <!-- contenu -->
</html>

À retenir : Toujours obligatoire !

1.2. La balise <html>

Utilité : La racine du document. Tout le code HTML est contenu à l'intérieur.

Attribut important : lang (indique la langue de la page)

Exemple :

<html lang="fr">
    <!-- tout le contenu -->
</html>

À retenir : La balise qui englobe TOUTE la page.

1.3. La balise <head>

Utilité : L'en-tête de la page. Contient les métadonnées (non visibles à l'écran).

Exemple :

<head>
    <meta charset="UTF-8">
    <title>Mon Site</title>
    <link rel="stylesheet" href="style.css">
</head>

À retenir : Ce qui est dans <head> ne s'affiche pas sur la page.

1.4. La balise <body>

Utilité : Le corps de la page. Tout ce qui est visible par l'utilisateur.

Exemple :

<body>
    <h1>Bienvenue sur mon site</h1>
    <p>Contenu visible</p>
</body>

À retenir : Tout ce que tu vois dans le navigateur est dans <body>.

1.5. La balise <title>

Utilité : Le titre de la page (s'affiche dans l'onglet du navigateur).

Emplacement : Dans <head>.

Exemple :

<title>Mon Site Web - Accueil</title>

À retenir : Très important pour le référencement (SEO).

1.6. La balise <meta>

Utilité : Définit des métadonnées (informations sur la page).

Attributs importants :

  • charset : encodage des caractères
  • name="viewport" : adaptation aux mobiles
  • name="description" : description pour les moteurs de recherche
  • name="keywords" : mots-clés

Exemple :

<!-- Encodage (pour les accents) -->
<meta charset="UTF-8">

<!-- Responsive (adaptation mobile) -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- Description (SEO) -->
<meta name="description" content="Description de mon site">

À retenir : Aident les moteurs de recherche et améliorent l'expérience utilisateur.

1.7. La balise <link>

Utilité : Lie une ressource externe (feuille de style CSS, police, icône, etc.).

Attributs importants :

  • rel : la relation avec la ressource (stylesheet, icon, etc.)
  • href : le chemin vers la ressource

Exemple :

<!-- Lien vers une feuille de style CSS -->
<link rel="stylesheet" href="style.css">

<!-- Lien vers une icône de favoris (favicon) -->
<link rel="icon" href="favicon.ico">

À retenir : Indispensable pour lier le CSS à ta page.

CHAPITRE 2 : LES BALISES DE TEXTE ET DE CONTENU

← Retour au Sommaire

Ces balises permettent de structurer et mettre en forme le texte.

2.1. Les titres <h1> à <h6>

Utilité : Définir des titres hiérarchiques (du plus important au moins important).

Balise Niveau Importance
<h1> Titre principal Très important (un seul par page)
<h2> Sous-titre Important
<h3> Sous-section Moyen
<h4> Sous-sous-section Secondaire
<h5> Titre de détail Peu important
<h6> Le plus petit Très peu important

Exemple :

<h1>Titre principal de la page</h1>
<h2>Section principale</h2>
<h3>Sous-section</h3>
<h4>Détail de la sous-section</h4>

À retenir : Le <h1> est unique par page pour le SEO.

2.2. Le paragraphe <p>

Utilité : Définir un paragraphe de texte.

Exemple :

<p>Ceci est un paragraphe normal.</p>
<p>Un autre paragraphe, séparé par un espace vertical.</p>

À retenir : La balise de base pour écrire du texte.

2.3. Les balises de mise en forme

Utilité : Modifier l'apparence du texte à l'intérieur d'un paragraphe.

Balise Effet Exemple
<strong> Texte important (gras) <strong>Important</strong>
<em> Texte mis en valeur (italique) <em>Valeur</em>
<b> Texte en gras (sans importance) <b>Gras</b>
<i> Texte en italique (sans importance) <i>Italique</i>
<u> Texte souligné <u>Souligné</u>
<s> Texte barré <s>Barré</s>
<mark> Texte surligné <mark>Surligné</mark>
<small> Texte plus petit <small>Petit</small>
<del> Texte supprimé <del>Supprimé</del>
<ins> Texte inséré <ins>Inséré</ins>

Exemple :

<p>
    Ceci est un <strong>texte important</strong>, 
    <em>mis en valeur</em>, 
    et <mark>surligné</mark>.
</p>

À retenir : Utilise <strong> et <em> pour leur sens sémantique.

2.4. Le saut de ligne <br>

Utilité : Faire un retour à la ligne (saut de ligne).

Exemple :

<p>
    Première ligne<br>
    Deuxième ligne<br>
    Troisième ligne
</p>

À retenir : <br> est une balise auto-fermante.

2.5. La ligne horizontale <hr>

Utilité : Créer une ligne de séparation horizontale.

Exemple :

<p>Contenu avant la ligne</p>
<hr>
<p>Contenu après la ligne</p>

À retenir : <hr> est une balise auto-fermante.

2.6. Le texte préformaté <pre>

Utilité : Afficher le texte tel quel (respecte les espaces et les retours à la ligne).

Exemple :

<pre>
    Dans un texte normal,
    les espaces    sont ignorés.
    Mais dans un   &lt;pre&gt;,
    ils  sont  conservés !
</pre>

À retenir : Idéal pour afficher du code ou de la poésie.

2.7. Le code <code>

Utilité : Afficher du code informatique (avec une police spéciale).

Exemple :

<p>La fonction <code>console.log()</code> affiche un message.</p>

À retenir : Souvent utilisé avec <pre> pour du code sur plusieurs lignes.

2.8. Les citations <blockquote> et <q>

Utilité : Afficher des citations.

  • <blockquote> : Citation longue (bloc)
  • <q> : Citation courte (en ligne)

Exemple :

<blockquote>
    La vie est comme une bicyclette, il faut avancer pour ne pas perdre l'équilibre.
    <footer>— Albert Einstein</footer>
</blockquote>

<p>Comme le dit <q>La simplicité est la sophistication suprême</q>.</p>

2.9. L'abréviation <abbr>

Utilité : Définir une abréviation ou un acronyme.

Exemple :

<p>
    Le <abbr title="HyperText Markup Language">HTML</abbr> 
    est le langage du Web.
</p>

À retenir : L'attribut title affiche la signification au survol.

2.10. L'adresse <address>

Utilité : Afficher des coordonnées (contact, adresse physique, email, etc.).

Exemple :

<address>
    Écrire à : <a href="mailto:contact@mon-site.com">contact@mon-site.com</a><br>
    Téléphone : 01 23 45 67 89<br>
    123 Rue de la République, Paris
</address>

CHAPITRE 3 : LES BALISES DE LISTES ET TABLEAUX

← Retour au Sommaire

3.1. Les listes non ordonnées <ul> et <li>

Utilité : Créer une liste à puces (points, cercles, carrés).

Exemple :

<ul>
    <li>Pommes</li>
    <li>Bananes</li>
    <li>Oranges</li>
</ul>

À retenir : <ul> = Unordered List, <li> = List Item.

3.2. Les listes ordonnées <ol> et <li>

Utilité : Créer une liste numérotée (chiffres, lettres, romains).

Exemple :

<ol>
    <li>Étape 1 : Ouvrir le frigo</li>
    <li>Étape 2 : Prendre les ingrédients</li>
    <li>Étape 3 : Manger</li>
</ol>

À retenir : <ol> = Ordered List.

3.3. Les listes de définitions <dl>, <dt>, <dd>

Utilité : Créer une liste de termes et leurs définitions.

Exemple :

<dl>
    <dt>HTML</dt>
    <dd>Langage de balisage pour créer des pages web</dd>
    <dt>CSS</dt>
    <dd>Langage pour styliser les pages web</dd>
</dl>

À retenir : <dl> = Definition List, <dt> = Definition Term, <dd> = Definition Description.

3.4. Les listes imbriquées

Utilité : Créer une liste dans une liste (sous-listes).

Exemple :

<ul>
    <li>Fruits
        <ul>
            <li>Pommes</li>
            <li>Bananes</li>
        </ul>
    </li>
    <li>Légumes
        <ul>
            <li>Carottes</li>
            <li>Haricots</li>
        </ul>
    </li>
</ul>

3.5. Les tableaux <table>

Utilité : Créer un tableau pour organiser des données.

Balise Utilité
<table> Crée un tableau
<thead> En-tête du tableau (première ligne)
<tbody> Corps du tableau (les données)
<tfoot> Pied du tableau (totaux, etc.)
<tr> Une ligne du tableau
<th> Cellule d'en-tête (en gras)
<td> Cellule de données

Exemple :

<table>
    <thead>
        <tr>
            <th>Nom</th>
            <th>Âge</th>
            <th>Ville</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Alice</td>
            <td>25</td>
            <td>Paris</td>
        </tr>
        <tr>
            <td>Bob</td>
            <td>30</td>
            <td>Lyon</td>
        </tr>
    </tbody>
</table>

Fusion de cellules

  • colspan : Fusionne plusieurs colonnes
  • rowspan : Fusionne plusieurs lignes

Exemple :

<td colspan="2">Je prends 2 colonnes</td>
<td rowspan="2">Je prends 2 lignes</td>

CHAPITRE 4 : LES BALISES DE LIENS ET MULTIMÉDIA

← Retour au Sommaire

4.1. Les liens <a>

Utilité : Créer un lien hypertexte vers une autre page, un fichier, une ancre, etc.

Attributs importants :

  • href : La destination du lien
  • target : Où ouvrir le lien (_blank = nouvel onglet)
  • title : Info-bulle au survol

Exemples :

<!-- Lien vers une page externe -->
<a href="https://www.google.com">Aller sur Google</a>

<!-- Lien vers une page interne -->
<a href="contact.html">Contact</a>

<!-- Lien vers une ancre (section de la page) -->
<a href="#section1">Aller à la Section 1</a>

<!-- Lien qui s'ouvre dans un nouvel onglet -->
<a href="https://google.com" target="_blank">Google (nouvel onglet)</a>

<!-- Lien pour envoyer un email -->
<a href="mailto:mon@email.com">Envoyer un email</a>

<!-- Lien pour appeler un numéro de téléphone -->
<a href="tel:+33123456789">Appeler</a>

4.2. Les images <img>

Utilité : Afficher une image.

Attributs importants :

  • src : Le chemin vers l'image
  • alt : Texte alternatif (accessibilité) - Obligatoire
  • width / height : Dimensions en pixels

Exemple :

<img src="mon-image.jpg" alt="Description de l'image">

Image avec légende

<figure>
    <img src="photo.jpg" alt="Un paysage de montagne">
    <figcaption>Les Alpes en été</figcaption>
</figure>

4.3. La vidéo <video>

Utilité : Intégrer une vidéo dans la page.

Attributs importants :

  • controls : Affiche les boutons de contrôle
  • autoplay : Lecture automatique ( souvent bloqué )
  • loop : Lecture en boucle
  • muted : Son coupé (obligatoire pour autoplay)
  • poster : Image d'aperçu avant la lecture

Exemple :

<video controls width="640" height="360">
    <source src="ma-video.mp4" type="video/mp4">
    <source src="ma-video.webm" type="video/webm">
    <p>Votre navigateur ne supporte pas la vidéo.</p>
</video>

4.4. L'audio <audio>

Utilité : Intégrer un fichier audio.

Exemple :

<audio controls>
    <source src="musique.mp3" type="audio/mpeg">
    <source src="musique.ogg" type="audio/ogg">
    <p>Votre navigateur ne supporte pas l'audio.</p>
</audio>

4.5. L'iframe <iframe>

Utilité : Intégrer une autre page web (YouTube, Google Maps, etc.).

Attributs importants :

  • src : L'URL de la page à intégrer
  • width / height : Dimensions
  • allowfullscreen : Autorise le plein écran

Exemples :

<!-- Intégrer une vidéo YouTube -->
<iframe 
    width="560" 
    height="315" 
    src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
    title="Vidéo YouTube" 
    allowfullscreen>
</iframe>

<!-- Intégrer une carte Google Maps -->
<iframe 
    src="https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!..." 
    width="600" 
    height="450" 
    style="border:0;" 
    allowfullscreen="" 
    loading="lazy">
</iframe>

4.6. Le canvas <canvas>

Utilité : Zone de dessin programmable avec JavaScript.

Exemple :

<canvas id="monCanvas" width="400" height="300"></canvas>

<script>
    const canvas = document.getElementById('monCanvas');
    const ctx = canvas.getContext('2d');
    ctx.fillStyle = 'red';
    ctx.fillRect(50, 50, 100, 100);
</script>

CHAPITRE 5 : LES BALISES DE FORMULAIRES

← Retour au Sommaire

5.1. Le formulaire <form>

Utilité : Créer un formulaire pour envoyer des données au serveur.

Attributs importants :

  • action : Où envoyer les données
  • method : Comment envoyer (GET ou POST)

5.2. Les champs de formulaire

Champ de texte <input type="text">

<label for="nom">Nom :</label>
<input type="text" id="nom" name="nom" placeholder="Votre nom" required>

Champ de mot de passe <input type="password">

<label for="mdp">Mot de passe :</label>
<input type="password" id="mdp" name="mdp">

Zone de texte <textarea>

<label for="message">Message :</label>
<textarea id="message" name="message" rows="5" cols="30"></textarea>

Boutons radio (choix unique) <input type="radio">

<fieldset>
    <legend>Genre :</legend>
    <label><input type="radio" name="genre" value="homme"> Homme</label>
    <label><input type="radio" name="genre" value="femme"> Femme</label>
    <label><input type="radio" name="genre" value="autre"> Autre</label>
</fieldset>

Cases à cocher (choix multiples) <input type="checkbox">

<label><input type="checkbox" name="loisirs" value="lecture"> Lecture</label>
<label><input type="checkbox" name="loisirs" value="sport"> Sport</label>
<label><input type="checkbox" name="loisirs" value="musique"> Musique</label>

Menu déroulant <select> et <option>

<label for="pays">Pays :</label>
<select id="pays" name="pays">
    <option value="">-- Choisissez un pays --</option>
    <option value="fr">France</option>
    <option value="be">Belgique</option>
    <option value="ch">Suisse</option>
</select>

Bouton de soumission <input type="submit"> et <button>

<input type="submit" value="Envoyer le formulaire">

<button type="submit">Envoyer le formulaire</button>

<button type="reset">Effacer tout</button>

5.3. Les nouveaux champs HTML5

Type Utilité Exemple
email Adresse email (validation automatique) <input type="email">
tel Numéro de téléphone <input type="tel">
url Adresse URL <input type="url">
number Nombre (avec min/max) <input type="number" min="0" max="120">
date Date (calendrier) <input type="date">
time Heure <input type="time">
color Couleur (sélecteur de couleur) <input type="color">
file Fichier (upload) <input type="file" accept="image/*">
search Recherche <input type="search">

CHAPITRE 6 : LES BALISES SÉMANTIQUES (HTML5)

← Retour au Sommaire

Ces balises donnent du sens à la structure de la page. Elles aident les moteurs de recherche, les lecteurs d'écran et les développeurs à comprendre l'organisation de la page.

6.1. <header>

Utilité : L'en-tête de la page ou d'une section. Contient le logo, le titre, la navigation.

<header>
    <h1>Mon Site Web</h1>
    <nav>...</nav>
</header>

6.2. <nav>

Utilité : Le menu de navigation principal.

<nav>
    <ul>
        <li><a href="#">Accueil</a></li>
        <li><a href="#">À propos</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>

6.3. <main>

Utilité : Le contenu principal de la page. Un seul par page.

<main>
    <h1>Titre de la page</h1>
    <p>Contenu principal...</p>
</main>

6.4. <section>

Utilité : Une section thématique de la page.

<section>
    <h2>Nos services</h2>
    <p>Description des services...</p>
</section>

6.5. <article>

Utilité : Un contenu indépendant et réutilisable (article de blog, news, etc.).

<article>
    <h2>Mon premier article</h2>
    <p>Contenu de l'article...</p>
    <footer>Publié le 17 juin 2026</footer>
</article>

6.6. <aside>

Utilité : Un contenu secondaire (sidebar, publicité, etc.).

<aside>
    <h3>Publicité</h3>
    <p>Venez découvrir nos offres !</p>
</aside>

6.7. <footer>

Utilité : Le pied de page (crédits, mentions légales, etc.).

<footer>
    <p>&copy; 2026 - Tous droits réservés</p>
</footer>

6.8. <figure> et <figcaption>

Utilité : Grouper une illustration (image, graphique) avec sa légende.

<figure>
    <img src="graphique.jpg" alt="Graphique des ventes">
    <figcaption>Ventes 2026 - Augmentation de 25%</figcaption>
</figure>

CHAPITRE 7 : LES BALISES DE MÉTADONNÉES

← Retour au Sommaire

Ces balises se placent dans <head> et ne sont pas visibles à l'écran.

7.1. <meta>

Utilité : Définir des métadonnées sur la page.

<meta charset="UTF-8">
<meta name="description" content="Description de mon site">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

7.2. <link>

Utilité : Lier une ressource externe (CSS, favicon, etc.).

<link rel="stylesheet" href="style.css">
<link rel="icon" href="favicon.ico">

7.3. <style>

Utilité : Définir du CSS directement dans la page.

<style>
    body { background-color: #f0f0f0; }
    h1 { color: blue; }
</style>

À retenir : Mieux vaut utiliser un fichier CSS externe (<link>) pour les gros projets.

7.4. <script>

Utilité : Ajouter du JavaScript à la page.


<script>
    console.log("Bonjour le monde !");
</script>
<!-- Ou en externe -->
<script src="script.js"></script>

7.5. <base>

Utilité : Définir l'URL de base pour tous les liens relatifs.

<base href="https://mon-site.com/">
